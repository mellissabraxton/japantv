By connecting WooCommerce shop with TrackingMore, merchants can track all orders effectively and improve customer’s satisfaction by automatically sending update emails to customers when order status updated.


== Description ==

###TrackingMore Parcel Tracking Plugin
After installing this plugin, TrackingMore system will auto import and track orders from WooCommerce shop every eight hours. After an order is fulfilled, customer can track his order at order history page by entering a tracking number. Currently, TrackingMore supports parcel tracking of 580 couriers worldwide. [Sign up free for a TrackingMore account](https://www.trackingmore.com/signup.html) and select your frequently used couriers.

###Manage Order Tracking Info Easily
This plugin alows you to display and store tracking info all in one place so that you can easily know your shipping performance! You can also filter orders by add time, carriers and order statuses.

###Track Button
TrackingMore [Track Button](https://www.trackingmore.com/embed_box_float.html) enables customers to track packages on their own by entering a tracking number. Merchants with little knowledge about programming can easily embed Track Button on their WooCommerce Shop. 

###Email and SMS Notifications of Delivery
After triggering notification settings, TrackingMore will auto send email or sms notifications when order status is in "Transit", "Out for Delivery", "Exception", "Delivered" and so on. Therefore, both merchants and customers can know the shipping condition in time and take measures before it is too late.

== Installation ==

1. [Sign up](https://www.trackingmore.com/signup.html) TrackingMore account for FREE.
2. Upload [TrackingMore plugin](https://downloads.wordpress.org/plugin/trackingmore-woocommerce-tracking.zip), then install and activate it at wordpress.
3. Go to Settings>TrackingMore, select couriers, use track button and then click save.
4. Generate TrackingMore's WordPress API Key under Users > Your Profile.
5. Enter Store URL and API Key you generated in [TrackingMore WooCommerce Connector page](https://www.trackingmore.com/apps-woocommerce.html).

[DEMO](https://www.youtube.com/watch?time_continue=1&v=HA_IrrHA79s)


*** This plugin requires at least PHP 5.2.4
*** This plugin requires at least WooCommerce 2.1
